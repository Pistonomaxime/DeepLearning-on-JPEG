  • valid – All the values that the user can choose.

Returns

User selected value.

Search if the Mnist and Cifar-10 directories already exists.

Returns

The directories qualities which already exist.

main.give_dataset()¶

Ask to the user to choose between Mnist and Cifar-10 datasets.

Returns

Choosen dataset.

main.give_quality(table_possible)¶

table_possible – Table of possible quality we can choose.

main.give_remaining_qualities(dataset, mnist_qualities, cifar_qualities)¶

From the input output a table of qualities the user can choose.

Parameters

  • dataset – Choosen dataset 0 for Mnist and 1 for Cifar-10.

  • mnist_qualities – Mnist directories qualities which already exist.

  • cifar_qualities – Cifar-10 directories qualities which already exist.

Raises

keyError – For the choosen dataset all qualities were already created.

Returns

Possible qualities the user can choose

Deeplearning file¶

deeplearning.deeplearning(quality, dataset, possible_steps, algorithm)¶

Load the selected dataset then compute the selection ML algorithm on this dataset.

Parameters

  • quality – Choosent JPEG quality between 100, 90, 80, 70 and 60.

  • dataset – Choosen dataset 0 for Mnist and 1 for Cifar-10.

  • possible_steps – Decompression step.

  • algorithm – Choosen ML algorithm.

Returns

Nothing

deeplearning.differents_noms(train_path, test_path, possible_steps)¶

Thanks to input output usefull information for computed ML algorithm.

Parameters

  • train_path – Train directory path.

  • test_path – Test directory path.

  • possible_steps – Decompression step.

Returns

Path to train and test file associated to decompression steps and ML save file name.
